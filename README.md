  Nightwatch end-to-end test automation boilerplate, using ES Modules, Babel and Assert.
  ## Getting Started
  	 git clone git@github.com:e2e-boilerplate/nightwatch-es-modules-babel-assert.git 
	 cd nightwatch-es-modules-babel-assert 
	 npm install 
	 npm run test
